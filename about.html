    Type Specimen

    — Design Practices:
          Print & Screen

    The aim of this project was to create a booklet and accompanying website for the promotion and sale of the typeface Ballinger. The core considerations was developing a core typographic system and using this in this in the application of print and screen UX/UI designs.

    The main design mentality for this project was to create a clear and functional layout to best show the merits and use cases for the Ballinger typeface this philosophy informed all my design decisions at all stages of the project. Minimal decorative elements were used in the layout so as not to distract from the main emphasis of the font itself.
